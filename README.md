<h3 id="about-description">
  ğŸ§¾ Description
</h3>
<p>
  <strong>QLogicae Plotica</strong> is a Windows x64 C++ static runtime benchmarking library. With its customizable file configurations and matplotlib-like visualizations, we aim to reduce project build time and to provide a medium for code runtime analysis - code less; more analysis.

</p>
